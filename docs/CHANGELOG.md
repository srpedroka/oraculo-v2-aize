# Changelog

- Concluida a Etapa 4 / Fatia 4B de CI obrigatorio, sem mudanca funcional no app ou WhatsApp. Pull requests e pushes para `main` passam por instalacao congelada, secret scan, audit, lint/typecheck, 162 unitarios, fixtures, build e smoke; um segundo job sobe Supabase local, aplica migrations, serve Edge Functions e executa integracao, RLS e E2E autenticado. Matriz documentada em `docs/TESTING.md`; sem migration, mudança visível ou deploy de produção.
